# HIV-TB Fractional Minimal Methods

Minimal reproducibility package containing only:

- `negative_binomial_estimation.py`: Negative Binomial likelihood and dispersion fitting utilities.
- `fractional_abm.py`: Fractional Adams-Bashforth-Moulton (ABM) Caputo solver utility.

This package intentionally excludes full project pipelines, plotting scripts, and manuscript-specific orchestration code.
